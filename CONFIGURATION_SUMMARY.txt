📋 USAGE EXAMPLES
================================================================================

Development (Already Configured):
  $ ./start-ollama.sh           # Terminal 1
  $ python app.py               # Terminal 2

Change Port:
  $ echo "PORT=8000" >> .env
  $ python app.py

Use Remote Ollama:
  $ echo "OLLAMA_HOST=ai.company.com" >> .env
  $ python app.py

Production Setup:
  $ cp .env.production.example .env
  $ nano .env                   # Edit values
  $ export FLASK_ENV=production
  $ python app.py

Docker:
  $ docker run -e FLASK_ENV=production \
               -e SECRET_KEY=<key> \
               astro-jyotish
